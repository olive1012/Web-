# Web-

GitHub Pagesプロジェクト - HTMLページをホストするためのリポジトリです。

## 📋 プロジェクト概要

このリポジトリはGitHub Pagesを使用してWebサイトをホストしています。

## 🚀 セットアップ手順

### GitHub Pagesの有効化

1. リポジトリの **Settings** に移動
2. 左サイドバーから **Pages** を選択
3. **Source** で以下を設定:
   - Branch: `main`
   - Folder: `/ (root)`
4. **Save** をクリック

### アクセス方法

セットアップ完了後、以下のURLでサイトにアクセスできます:
```
https://olive1012.github.io/Web-/
```

## 📁 ファイル構成

- `index.html` - メインページ
- `README.md` - このファイル

## 🎨 カスタマイズ

`index.html`を編集してページをカスタマイズできます。
